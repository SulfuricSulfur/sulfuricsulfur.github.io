Project 3: BrickBreaker

An HTML, CSS, and Javascript web game project for my Web Design and Implementation class.

Rules

  • Goal: Clear all the bricks off the screen using the ball to hit the bricks and the paddle to guide the ball.
  • Avoid: Having the ball go past the paddle. If this happens, you lose a life.
    • Once you lose all your lives the game is over
  • Every time you finish a level you will get a bonus and move onto next level.
  • Try to finish game with highscore.
  • Have fun!
